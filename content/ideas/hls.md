[Haskell Language Server](https://github.com/haskell/haskell-language-server) is
a full-featured Haskell IDE that recently reached version `v1.0`.  Since it is
a large project, there's a lot of possible ideas for students.

[This thread](https://github.com/haskell/haskell-language-server/discussions/1237) has
a discussions around projects, and there are a number of issues tagged as
[Eligible for GSoC 2021](https://github.com/haskell/haskell-language-server/labels/Eligible%20for%20gsoc%202021).

**Mentors**: the Haskell Language Server team
